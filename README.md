# 📰 TechNews - Portal de Notícias Tecnológicas

Um portal de notícias moderno focado no ecossistema de tecnologia, desenvolvido para consolidação de fundamentos em desenvolvimento web (HTML estrutural e estilização CSS). O projeto foca no domínio de estruturas semânticas, organização de layout e posicionamento de elementos em tela.

🔗 **[Acesse o projeto publicado aqui](https://gustavogoncalveslentz.github.io/portal-de-noticias/)**

---

## 🎯 Objetivo e Escopo

Este projeto foi construído com o objetivo focado no domínio de técnicas estruturais de front-end. A proposta consiste em replicar a interface de um portal de notícias com temas como Inteligência Artificial, Realidade Virtual, Blockchain e Robótica.

O desenvolvimento focou especificamente em:
- **HTML5 Semântico:** Uso de tags como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>` e `<figure>` para garantir boa estrutura e acessibilidade.
- **CSS (Layout & Posicionamento):** Aplicação prática de Flexbox e Grid Layout para organizar a distribuição dos blocos de conteúdo e a barra lateral (*aside*).

## 🛠️ Tecnologias Utilizadas

Por ser um projeto com escopo voltado para a fundamentação estrutural e visual estática, foram utilizadas as tecnologias nativas da web:

*   **HTML:** Estruturação semântica de todo o conteúdo da página.
*   **CSS:** Estilização global, tipografia personalizada (Google Fonts), cores e gerenciamento do layout da página.

*Nota: Este projeto trata-se exclusivamente de uma interface visual estática, sem comportamento dinâmico ou regras de lógica de programação.*
